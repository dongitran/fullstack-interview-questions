# Fullstack Interview Questions

A clean, mobile-first Q&A site for **Senior Fullstack Developer** interviews — focused on **concepts, trade-offs, and real-world decision making**.

**Live:** https://dongitran.github.io/fullstack-interview-questions/

---

## What’s inside

- **70 questions** across **9 topics**
- Practical explanations (easy to understand)
- Covers: microservices, payments, databases & performance, Kafka/queues, system design, DevOps, observability, and more

## Quick start (local)

```bash
git clone https://github.com/dongitran/fullstack-interview-questions.git
cd fullstack-interview-questions
open index.html
```
